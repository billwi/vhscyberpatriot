# [vhscyberpatriot.tech](vhscyberpatriot.tech)

vhscyberpatriot is the (un)official website for Valencia's CyberPatriot group

## Contributing

 - PRs not accepted!
 - Approved people can follow these guidelines to contribute

## Adding Images/READMEs

 1. Create a blog post from the `_posts` directory
 2. Make sure the post contains the download link for the image and the readme link
 3. Add the readme in a gist format in the `ReadmeDump` folder
 4. Create a title for the image, and the download links seen in step `2` in the `readmes.md` page
